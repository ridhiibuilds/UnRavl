# UnRavl 🧵⚡

> AI powered wardrobe upcycling tool — transform 
clothes you're bored of into something you actually 
want to wear.

---

## The Problem

Most people have wardrobes full of clothes they don't 
wear anymore. Not because they're damaged — but because 
they're bored of them.

Buying new ones is wasteful. Throwing them away feels 
wrong. Pinterest gives you inspo but doesn't tell you 
how to execute it on YOUR specific piece.

Nobody has built a tool that looks at YOUR specific 
garment and says — here's exactly what you can do 
with this, today, with basic tools.

That's UnRavl.

---

## Features

- 🔍 Garment detection from photo
- ✂️ Specific upcycle suggestions by garment type
- 🎨 Pinterest inspo matching
- 💾 Save your upcycle mood board

---

## Tech Stack

- Python
- Streamlit
- HuggingFace Transformers
- Pillow / OpenCV
